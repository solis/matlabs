Дополнительная лабораторная №2

Уравнение
        Δ u = (-400) .* (cos(20 .* (y - 1) .* (x -1))) .* ((1-x).^2 + (1 - y).^2)
Граничные условия
        На вертикальных границах
            psi(y) = (-20 .* (-1+y) .* sin(20 .* (-1+x) .* (-1+y)))
        На горизонтальных границах
            phi(x) = cos(20(x - 1)(c - 1)), c in {0, 0.5, 0.9}
Точное решение
        u(x,y) = cos(20.*(x-1).*(y-1))
